# 🚀 DEPLOYMENT INSTRUCTIONS FOR COMPANY PORTAL

## Method 1: GitHub Pages (Recommended)

### Quick Start - 5 Minutes Setup

1. **Login to GitHub**
   - Go to https://github.com
   - Sign in with your account

2. **Create New Repository**
   - Click "+" icon (top right) → "New repository"
   - Repository name: `company-portal`
   - Description: "Midwest VG Solutions Company Portal"
   - Select: ✅ Public
   - Check: ✅ Add a README file
   - Click: "Create repository"

3. **Upload Files**
   - Click "Add file" → "Upload files"
   - Upload these files:
     * index.html (the main portal file)
     * README.md (documentation)
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Click "Settings" tab
   - Click "Pages" in left sidebar
   - Under "Source": Select "main" branch
   - Click "Save"
   - Wait 2-3 minutes

5. **Get Your Live URL**
   - Your portal will be live at:
   - `https://[your-github-username].github.io/company-portal/`
   - Example: `https://midwestvgsolutions.github.io/company-portal/`

6. **Test It**
   - Visit the URL
   - Try logging in with:
     - Company: Kishan Inc.
     - Password: vgsolutions2026

## Method 2: Custom Domain (Optional)

If you own a domain like `midwestvg.com`:

1. **In GitHub Pages Settings:**
   - Enter your custom domain (e.g., `portal.midwestvg.com`)
   - Click "Save"

2. **In Your Domain Registrar (GoDaddy, Namecheap, etc.):**
   - Add a CNAME record:
     - Name: `portal`
     - Value: `[your-username].github.io`
   - Save changes

3. **Wait 24 hours** for DNS to propagate

4. **Your portal will be at:** `https://portal.midwestvg.com`
